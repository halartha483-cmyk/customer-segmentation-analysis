# Customer Segmentation Analysis

## Tools Used

* Tableau
* Excel

## Project Overview

This project analyzes customer purchasing behavior by segmenting customers into High, Medium, and Low value groups based on total spending and order patterns.

## Key Metrics

* Total Spend per Customer
* Number of Orders
* Average Order Value
* Customer Segment Classification

## Dashboard Features

* Customers by Segment distribution
* Revenue by Segment comparison
* Average Order Value by Segment
* Top Customers by Spend

## Files Included

* Cleaned dataset
* Tableau dashboard file
* Dashboard export (PDF)
